# 回答质量评估维度

> 来源：Tech Interview Handbook (FAANG 通用), Polymer, 简道云, Yardstick

## 技术面试评分四维度

| 维度 | 差评信号 | 优秀信号 |
|------|---------|---------|
| **沟通能力** | 极难理解候选人思路 | 毫无困难跟随完整思考过程 |
| **问题解决** | 无法系统解决问题 | 讨论延伸问题，展现高级信号 |
| **技术能力** | 多个概念错误和糟糕实践 | 轻松展示基础和高级能力 |
| **测试能力** | 未考虑典型和边界情况 | 毫不费力展示测试信号 |

## 行为面试评分三维度

| 维度 | 说明 |
|------|------|
| 工作要求与能力 | 与角色直接相关的实际技能 |
| 软技能 | 批判性思维、问题解决、沟通、团队协作 |
| 核心价值观 | 候选人的优先级和信念与团队文化的契合度 |

## 五级评分制

| 分数 | 含义 |
|------|------|
| 1 | 无经验 |
| 2 | 有限经验 |
| 3 | 标准经验 |
| 4 | 多项广泛经验 |
| 5 | 广泛且杰出经验 |

## 候选人回答模式识别

### 背书式回答
特征：回答像在背课本，缺少个人经验和具体细节。
应对：追问具体实现细节和实际遇到的问题。

### 空泛式回答
特征：大量使用"优化了性能"、"提升了用户体验"等模糊表述。
应对：要求给出具体数据和实例。

### 防御式回答
特征：面对质疑时过度解释或转移话题。
应对：保持友善但坚持追问核心问题。

## 减少主观性的三大策略

1. **行为面试法（STAR）**：通过情境/任务/行动/结果四维度了解实际表现
2. **标准化工具**：统一的评分表使不同评估者对同一候选人的评价保持一致
3. **多维度评估**：职位匹配度、行为态度、问题解决能力、文化契合度、发展潜力
