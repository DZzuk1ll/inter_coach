# 压力面试技巧

> 来源：PMaps, MBA智库, Indeed

## 目的
压力面试不是为了刁难候选人，而是测试候选人在质疑和压力下的反应，考察技术自信度和自我认知，挖掘真实水平边界。

## 六种压力面试类型

1. **攻击性提问**：使用挑衅或对抗性问题挑战自信心
2. **敌意/恐吓型**：面试官表现冷漠、否定，频繁打断
3. **混淆/矛盾型**：提出相互矛盾或模糊的问题测试逻辑推理
4. **不适情境型**：在面试前就制造不便或意外
5. **角色扮演型**：模拟高压工作场景要求实时应对
6. **脑筋急转弯型**：使用逻辑题评估创造性思维

## 常用技巧

| 技巧 | 描述 |
|------|------|
| 激将法 | 从求职者薄弱处入手，使用质疑语言制造尴尬 |
| 诱导法 | 提供特定选择诱导错误回答，或制造思维陷阱 |
| 快速连环提问 | 快速接连提出挑战性问题，不给太多思考时间 |
| 长时间沉默 | 回答后故意长时间不回应，制造心理压力 |
| 反转假设 | "如果我告诉你 X 比 Y 更好，你怎么看？" |

## 评估的四个维度

1. **情绪智力**：能否控制自己的反应？
2. **沉着度**：如何应对不舒服或高压时刻？
3. **压力下的问题解决**：是反应型还是反思型？
4. **自信与坚定**：能否清晰沟通而不攻击性或退缩？

## 注意事项
- 压力追问比例不超过总面试的 20-25%
- 如果候选人已经明显紧张或沮丧，立即缓和气氛
- 质疑后如果候选人给出了合理解释，要认可
- 挑战候选人但不恐吓候选人，全程保持专业性
- 质疑要基于代码事实，不能无中生有
